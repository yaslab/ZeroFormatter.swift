## Supported types (Stage1 only)

### Primitive Format

| Implemented | Swift type | C# type |
| ---- | ---- | ---- |
| o | `Int8` | `SByte` |
| o | `Int16` | ← |
| o | `Int32` | ← |
| o | `Int64` | ← |
| o | `UInt8` | `Byte` |
| o | `UInt16` | ← and `Char` |
| o | `UInt32` | ← |
| o | `UInt64` | ← |
| o | `Float` | `Single` |
| o | `Double` | ← |
| o | `Bool` | `Boolean` |
| | TBD | `TimeSpan` |
| o | `Date` | `DateTime` |
| | TBD | `DateTimeOffset` |
| o | `String` | ← |
